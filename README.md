# Satellite-Based-Greenwashing-Detection
Satellite Based Greenwashing Detection analyzes satellite images using machine learning to detect possible greenwashing by evaluating vegetation health and environmental indicators. Results are shown in a Streamlit dashboard.
The Satellite Based Greenwashing Detection System is an AI-powered environmental monitoring solution that analyzes satellite imagery to detect potential greenwashing practices. Many organizations claim their projects are environmentally friendly, but these claims may not always reflect the actual environmental conditions.
This project uses satellite images, image processing, and machine learning techniques to analyze vegetation health and environmental indicators across large geographical areas. The system calculates environmental metrics such as NDVI (Normalized Difference Vegetation Index) and vegetation coverage to evaluate environmental sustainability.
The results are displayed through an interactive Streamlit dashboard that provides vegetation heatmaps, environmental indicators, and a Greenwashing Risk Score to determine whether sustainability claims match the actual environmental conditions.
